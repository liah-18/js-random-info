JavaScript – Random Information from an Array

Each time you click the button, JavaScript will randomly select one of my cats from an array and display the name on the page.

When the button is clicked, it runs a function called showInfo(). That function generates a random number, uses it to choose one item from the array, and then places the result inside the element with the ID info.

This exercise demonstrates how these pieces work together:

  • An array stores several possible values.
  • A function runs when the button is clicked.
  • A random number is used to choose one item from the array.
  • JavaScript updates the page by changing the content inside an HTML element.
